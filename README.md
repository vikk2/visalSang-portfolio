# Minimal CS Portfolio

Simple static portfolio for a third-year Computer Science student.

## Pages

- `index.html`
- `about.html`
- `projects.html`
- `contact.html`

## Notes

- Edit the placeholder name, bio, email, GitHub, LinkedIn, and project content.
- The token saver on the contact page stores data only in your local browser with `localStorage`.
- Open `index.html` directly in a browser, or serve the folder with a static server if you prefer.
